## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config-defense/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Расширение модуля «Настройка доступа по IP-адресу»

Настройка правил доступа к Панели управления GePanel и веб‑приложению RosGear на основе фильтрации по IP‑адресам.

Расширение модуля входит в состав Панели управления GePanel и cлужит расширением для [модуля «Конфигурация»](https://github.com/rosgear/rg-be-config).

### Скриншот
<img src="https://github.com/rosgear/rg-be-config-defense/blob/main/assets/help/form.png?raw=true">

### Установка

Для добавления расширения модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-config-defense
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-config-defense": "*"
}
```
или скачать архив на [странице расширения модуля](https://rosgear.ru/component/rg-be-config-defense/) в каталоге приложений RosGear.

После добавления расширения модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

<a name="english"></a>
## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config-defense/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module extension «Configuring access by IP Address»

Configure access rules to the GePanel Control Panel and the RosGear web application based on IP address filtering.

The module extension is part of the GePanel Control Panel and serves as an extension for the [«Configuration» module](https://github.com/rosgear/rg-be-config).

### Screenshot
<img src="https://github.com/rosgear/rg-be-config-defense/blob/main/assets/help/form.png?raw=true">

### Installation

To add the module extension to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-config-defense
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-config-defense": "*"
}
```
or download the archive from the [module extension page](https://rosgear.ru/component/rg-be-config-defense/) in the RosGear application catalog.

After adding the module extension to the project, install it into the web application edition using the GePanel Control Panel.
